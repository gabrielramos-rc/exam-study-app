```tsx
interface OptionButtonProps {
  label: string;        // "A", "B", etc.
  text: string;         // Option text
  selected: boolean;
  correct?: boolean;    // Only when showing answer
  showResult: boolean;
  onClick: () => void;
  disabled: boolean;
}
```

States:
- Default: Gray border
- Hover: Blue border
- Selected: Blue background
- Correct (revealed): Green background
- Wrong (revealed): Red background
